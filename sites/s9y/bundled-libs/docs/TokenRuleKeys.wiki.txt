++ Tokens Entry Format

Simple format (no options).

<code>
$_tokens[] = Array
    (
        [0] => 'rule_name'
        [1] => array
            (
            )
    )
</code>

Complex format (with options).

<code>
$_tokens[] = Array
    (
        [0] => 'rule_name'
        [1] => Array
            (
                ['opt0'] => 'value 0'
                ['opt1'] => 'value 1'
                ...
                ['optN'] => 'value N'
            )
    )
</code>
